# Vibe Recipe Generator Roadmap

An evolving plan for features and improvements to the vibe recipe generator.

## Current State

- AI-powered recipe generation from vibe/mood descriptions
- React 19 + TypeScript frontend with CSS Modules
- Express backend with Claude Sonnet
- Recipe card with ingredients, steps, vibe notes
- Error handling, retry button, rate limit countdown
- Escape key navigation and focus management

## Ideas Under Consideration

- **Save/favorite recipes** — persist generated recipes locally or to a backend
- **Recipe history** — browse previously generated recipes
- **Share recipes** — shareable links or copy-to-clipboard
- **Vibe suggestions** — prompt chips to inspire input (e.g. "cozy rainy day", "summer BBQ")
- **Dietary filters** — pass constraints (vegan, gluten-free) alongside the vibe
- **Image generation** — AI-generated dish photo to accompany the recipe
- **Print-friendly view** — clean recipe printout
